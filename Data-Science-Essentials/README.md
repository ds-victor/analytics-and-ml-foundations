# 🤖 Data Science Essentials

## 1. Header & Quick Overview
Welcome to the **Data Science Essentials** module! This repository focuses on the end-to-end data science workflow, using a **House Price Prediction** project as our running example. This module is designed to bridge the gap between understanding data and deriving actionable business value through Machine Learning.

## 2. Who Should Take This?
- **Beginners** transitioning from data analytics to data science.
- **Career Changers** looking to understand the professional data science lifecycle.

## 6. Getting Started
1. **Setup environment:**
   ```bash
   pip install -r requirements.txt
   ```
2. **Verify Installation:**
   Run `python scripts/problem_framing_demo.py`.

## 7. How to Use This Repository
- **`/topics`**: Markdown files for conceptual learning.
- **`/scripts`**: Executable Python code for production-like workflows.
- **`/notebooks`**: Interactive Jupyter notebooks for EDA and exploration.
- **`/data`**: Contains the house price dataset.
