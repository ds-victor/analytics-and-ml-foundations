# Final Quiz: ML Foundations Series

Test your understanding of basic machine learning concepts, paradigms, and model evaluation.

## Questions

1. **Which machine learning paradigm involves learning from labeled data?**
   - A) Unsupervised Learning
   - B) Reinforcement Learning
   - C) Supervised Learning
   - D) Clustering

2. **In a regression task, what type of value are you trying to predict?**
   - A) A discrete category (e.g., "Spam" or "Not Spam").
   - B) A continuous numerical value (e.g., house price).
   - C) A sequence of actions.
   - D) A cluster ID.

3. **What does 'Overfitting' mean in machine learning?**
   - A) The model is too simple to capture the underlying pattern.
   - B) The model has learned the noise in the training data too well and performs poorly on unseen data.
   - C) The model training process took too long.
   - D) The dataset is too small for the model.

4. **Which of the following is a common evaluation metric for classification tasks?**
   - A) Mean Squared Error (MSE)
   - B) R-squared
   - C) F1-Score
   - D) Mean Absolute Error (MAE)

5. **Why do we use a 'Random State' (seed) when splitting data?**
   - A) To make the code run faster.
   - B) To ensure the results are reproducible by getting the same split every time.
   - C) To improve the accuracy of the model.
   - D) To automatically handle missing values.

6. **The 'Bias-Variance Trade-off' refers to the balance between:**
   - A) Training time and prediction time.
   - B) Model complexity and its ability to generalize to new data.
   - C) The cost of data and the value of the model.
   - D) SQL queries and Python scripts.

7. **Which algorithm is an example of an 'Ensemble Method'?**
   - A) Linear Regression
   - B) Logistic Regression
   - C) Random Forest
   - D) K-Means Clustering

8. **In 'K-fold Cross-Validation', the data is:**
   - A) Split once into 80% train and 20% test.
   - B) Divided into K subsets, and the model is trained/validated K times.
   - C) Only used for unsupervised learning.
   - D) Deleted after the first fold.

9. **What is the primary purpose of a 'Validation Set'?**
   - A) To provide the final performance metric for the model.
   - B) To tune hyperparameters and prevent overfitting to the training set.
   - C) To replace the training set entirely.
   - D) To store the raw data.

10. **Streamlit is primarily used by data scientists to:**
    - A) Manage database servers.
    - B) Write deep learning frameworks from scratch.
    - C) Build and deploy interactive web applications for their models.
    - D) Clean large datasets in SQL.
