# Contributing to Data Science Essentials

Welcome! We're glad you're here.

## How to Contribute
- **Report Issues:** Use GitHub Issues to report bugs or suggest enhancements.
- **Propose Improvements:** Fork the repo, create a branch, and submit a pull request.
- **Code Style:** Follow PEP 8 guidelines and use `black` for formatting.
- **New Examples:** When adding examples, please use the provided `real_estate_dataset.csv` or document any new data sources clearly.

## Review Process
All pull requests will be reviewed by the maintainers. We aim to provide feedback within a few business days.

## Code of Conduct
Please be inclusive and respectful in all your interactions.
