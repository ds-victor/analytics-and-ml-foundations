## 📌 Overview

This repository is dedicated to the **Data Analyst** path. It is a structured, project-based learning module designed to help you develop job-ready Data Analytics skills through practical exercises, guided lessons, and Exploratory Data Analysis (EDA) using a realistic real-estate dataset.

## 📂 Independent Structure

This sub-repository is entirely independent. It contains everything you need for Data Analytics:
- **`data/raw/real_estate_dataset.csv`**: The dataset for this module.
- **`topics/`**: Detailed markdown lessons organized into learning levels.
- **`notebooks/`**: An end-to-end EDA notebook.
- **`scripts/`**: Automated reporting scripts.

## 🚀 Quick Start

1. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Explore the EDA Notebook:**
   ```bash
   jupyter notebook notebooks/01_eda_analysis.ipynb
   ```

3. **Run the Automated Report:**
   ```bash
   python scripts/eda_report.py
   ```
