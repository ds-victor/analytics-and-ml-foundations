# Final Quiz: Data Analytics Essentials

Test your knowledge of SQL, Python for data analysis, and Exploratory Data Analysis (EDA).

## Questions

1. **Which SQL clause is used to filter rows based on a specific condition?**
   - A) SELECT
   - B) GROUP BY
   - C) WHERE
   - D) ORDER BY

2. **In Pandas, which function is used to get a summary of a DataFrame, including data types and non-null counts?**
   - A) `df.describe()`
   - B) `df.info()`
   - C) `df.head()`
   - D) `df.summary()`

3. **What is the primary purpose of Exploratory Data Analysis (EDA)?**
   - A) To build a final production-ready machine learning model.
   - B) To understand data patterns, anomalies, and relationships before deeper analysis.
   - C) To only create beautiful charts for a presentation.
   - D) To delete all missing values from the dataset.

4. **Which chart is best for showing the relationship between two continuous numerical variables?**
   - A) Bar Chart
   - B) Histogram
   - C) Scatter Plot
   - D) Pie Chart

5. **In a SQL query, which keyword is used to combine rows from two or more tables based on a related column?**
   - A) MERGE
   - B) JOIN
   - C) UNION
   - D) CONNECT

6. **A 'Boxplot' is particularly useful for identifying:**
   - A) Linear trends over time.
   - B) Outliers and the spread of data.
   - C) Categorical proportions.
   - D) SQL syntax errors.

7. **Which SQL aggregate function would you use to find the average value of a column?**
   - A) COUNT()
   - B) SUM()
   - C) AVG()
   - D) MEAN()

8. **What does the 'p-value' represent in statistical hypothesis testing?**
   - A) The probability that the hypothesis is true.
   - B) The probability of observing the data if the null hypothesis is true.
   - C) The power of the test.
   - D) The number of parameters in the model.

9. **In Python, which library is the standard for data manipulation and analysis?**
   - A) Matplotlib
   - B) Requests
   - C) Pandas
   - D) Flask

10. **When handling missing data, 'Imputation' refers to:**
    - A) Deleting all rows with missing values.
    - B) Replacing missing values with estimated ones (like the mean or median).
    - C) Ignoring the missing values entirely.
    - D) Asking the user to provide the missing data.
